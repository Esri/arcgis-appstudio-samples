
## Migration tool for Version 1.0 of AppStudio

This is a migration tool for those that have invested time and effort in beta versions of the Map Tour. With the final release of AppStudio version 1.0, a runtime import (Runtime.Dialogs) is no longer available. This tool will search all your Map tours and update them so they continue to work. This will only work on apps that have not had folder structure and file name changes.

## Instructions to run this sample in AppStudio Desktop

1. Download the `.zip` file
2. Unzip and copy this folder into AppStudio Apps folder (Windows: `C:\Users\<username>\ArcGIS\AppStudio\Apps` Mac or linux: `Home\ArcGIS\AppStudio\Apps`)
3. The new app will now appear in the AppStudio Desktop. Run the application or open it in the bundled Qt-Creator IDE to look at the code and modify.
