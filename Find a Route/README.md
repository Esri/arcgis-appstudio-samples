
## Calculate route between two locations

This sample demonstrates the following:
- For simplicity, the sample comes loaded with a start and end stop. 
- You can tap on the route button to get a route between these stops. Once the route is generated, the directions button gets enabled. 
- Tap on the button to see the step by step directions for the route.
- The sample uses the generateDefaultParameters method on RouteTask to get the default parameters from the service.

## Instructions to run this sample in AppStudio Desktop

1. Download the `.zip` file
2. Unzip and copy this folder into AppStudio Apps folder (Windows: `C:\Users\<username>\ArcGIS\AppStudio\Apps` Mac or linux: `Home\ArcGIS\AppStudio\Apps`)
3. The new app will now appear in the AppStudio Desktop. Run the application or open it in the bundled Qt-Creator IDE to look at the code and modify.
