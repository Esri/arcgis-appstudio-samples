## Web Map

This sample shows how to load a Web Map, show Web Map legend, and show Web Map popups.

There are 3 samples in this item:
- Show Web Map Legend: This sample demonstrates how to show a legend for all the layers within webmap
- Load Web Map: This sample demonstrates how to load a webmap in a MapView using Runtime. The Map initUrl property is defined using the webmap url. The webmap url is created using the webmap
- Web Map Identify: This sample demonstrates how to show webmap popups in a MapView using Runtime. The Map initUrl property is defined using the webmap url. The webmap url is created using the webmap

## Instructions to run this sample in AppStudio Desktop

1. Download the `.zip` file
2. Unzip and copy this folder into AppStudio Apps folder (Windows: `C:\Users\<username>\ArcGIS\AppStudio\Apps` Mac or linux: `Home\ArcGIS\AppStudio\Apps`)
3. The new app will now appear in the AppStudio Desktop. Run the application or open it in the bundled Qt-Creator IDE to look at the code and modify.
