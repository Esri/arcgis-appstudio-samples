
## Email Composer

This sample demonstrates how to use EmailComposer component in InterAppCommunication plugin.

The EmailComposer component allows you to launch a system email client with a pre-filled draft email, ready to send, modify, delete or save as a draft.

When your clicking on the button in the sample app, it will take a screenshot and generate a pre-filled email draft with the screenshot attachment.

## Instructions to run this sample in AppStudio Desktop

1. Download the `.zip` file
2. Unzip and copy this folder into AppStudio Apps folder (Windows: `C:\Users\<username>\ArcGIS\AppStudio\Apps` Mac or linux: `Home\ArcGIS\AppStudio\Apps`)
3. The new app will now appear in the AppStudio Desktop. Run the application or open it in the bundled Qt-Creator IDE to look at the code and modify.
